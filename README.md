# Usage
<!-- usage -->
```sh-session
$ npm install -g fancywow
$ fancywow COMMAND
running command...
$ fancywow (-v|--version|version)
fancywow/0.0.0 linux-x64 node-v8.11.3
$ fancywow --help [COMMAND]
USAGE
  $ fancywow COMMAND
...
```
